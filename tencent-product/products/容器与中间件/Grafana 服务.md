# Grafana 服务

Grafana 服务（TencentCloud Managed Service for Grafana，TCMG）是腾讯云基于社区广受欢迎的开源可视化项目 Grafana ，并与 Grafana Lab 合作开发的托管服务。TCMG 为您提供安全、免运维 Grafana 的能力，内建腾讯云多种数据源插件，如 Prometheus 监控服务、容器服务、日志服务 、Graphite 和 InfluxDB 等，最终实现数据的统一可视化。

[演示Demo](https://demo.grafana.tencent-cloud.com/d/dynWGDAIk/demo?orgId=1)客户交流群

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/UFohtPIaLvrjrCidwejJd.png)

### 安全可靠

TCMG 与腾讯云账号 SSO 进行整合，通过腾讯云的二次校验等鉴权能力，细粒度管控仪表板及监控数据安全。同时结合私有网络（Virtual Private Cloud，VPC）能力， 支持开通内、外网访问控制，进一步提升 Grafana 整体安全性。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/TWNb4tYeIFt0UXzcFdZJn.png)

### 多数据源统一可视化

预置多种腾讯云数据源插件，快速对接云产品监控、Prometheus 监控服务、日志服务和 Elasticsearch 服务等，同时保留了对开源社区第三方数据源（Graphite，InfluxDB等）的对接能力，实现集中展示和管理。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/fHSIAuTfNzoQDLYUAT-L8.png)

### 低成本管理

TCMG 自动为您构建、安装、部署、升级和管理 Grafana,节约运维成本。预建远程图片渲染服务，节约用户自行搭建服务器成本及管理成本。预设了云产品监控相关仪表板以及插件， 开箱即用。

- 云产品数据可视化
- 全栈监控可视化
- 日志数据可视化

通过 Grafana 服务 ，对接云上 Elasticsearch Service 及日志服务（CLS）， 实现日志数据的可视化。

### Grafana 服务与在云上自建 Grafana 有何区别， 有何优势？

Grafana 服务是腾讯云与 Grafana Lab 联合的发布的原生可视化解决方案， 预制了大量腾讯云相关相关数据源。同时利用 SSO 及网络访问控制，提升了数据安全性。使用托管服务可免除用户搭建，运维，升级等相关工作量。
