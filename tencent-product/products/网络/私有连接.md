# 私有连接 Private Link

私有连接帮助您实现在腾讯云 VPC 网络通过安全稳定的网络连接来访问其他 VPC 网络中的服务资源，简化网络架构、降低公网带宽，轻松实现更低成本、更安全、更灵活的网络连接服务。

### 高质量的服务共享

业务应用部署在腾讯云 VPC 内的云服务器上，以服务方式对其他 VPC 提供定向服务访问。同地域内网通信，不仅能节约公网带宽，有效降低带宽成本，且内网延时可控，通信质量更高。

![](https://qcloudimg.tencent-cloud.cn/raw/f05d25610aa0c09f338a1612613f46f6.png)

### 安全控制

基于腾讯云内网的安全访问，能够有效避免公网访问带来的潜在风险，大大提高数据的安全性。服务提供方可以精准控制使用方的连接请求，服务使用方可以利用安全组来进行访问控制，从而有效降低通信风险。

### 灵活配置及管理

服务使用方与提供方之间无需配置复杂的路由策略，即可建立安全的访问连接，结合控制台可视化的实时指标监控，提高管理效率。

- 共享云上应用服务
- 多网络访问云上服务

结合丰富的网络连接服务，可以实现 IDC、跨地域 VPC 等多网络均通过服务使用方的终端节点来访问服务提供方部署的应用服务。

### 私有连接可以实现跨地域的 VPC 访问么？

私有连接本身是提供同地域的跨 VPC 访问，如果需要跨地域实现 VPC 访问，可以结合云联网使用。
