# 弹性公网 IPv6

弹性公网 IPv6（Elastic Public IPv6，EIPv6）是您在腾讯云某个地域可以独立申请和持有的、固定不变的公网 IPv6 地址，提供与弹性公网 IP 一致的产品体验。您可以按需为云资源开通或关闭 IPv6 公网通信能力，灵活进行带宽峰值设置，满足业务需求。

![](https://qcloudimg.tencent-cloud.cn/raw/854fc7f9daebcd73df8e8d2948cbe73a.png)

### 安全灵活

您可以通过弹性公网 IPv6 为您的云资源快速开通或关闭 IPv6 公网通信能力；灵活设置 IPv6 公网带宽峰值；开通 IPv6 公网带宽后，将默认启用基础 DDoS 防护。

### 价格实惠

弹性公网 IPv6 支持多种计费模式，如按流量计费、按共享带宽包计费。同地域公网 IPv4、IPv6 流量、带宽资费标准保持一致，且 IPv6 地址不再单独收取 IP 资源费，可帮助您节约网络成本。

### 高可用

通过分布式网关集群、跨机架容灾、跨机房容灾、分布式控制器、底层架构能力以及灵活的扩展能力，弹性公网 IPv6 实现了整体架构的高可用和服务高可用。

电商、游戏等业务所在的云服务器同时运行 IPv6 和 IPv4，要求 IPv6 和 IPv4 都具备公网访问能力，且相互独立和相互备份。IPv6 支持独立设置公网带宽上限，且支持业务大带宽、低时延等要求。
