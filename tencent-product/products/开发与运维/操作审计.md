# 操作审计 CloudAudit

操作审计 CloudAudit 是一项支持对您的腾讯云账号进行监管、合规性检查、操作审核和风险审核的服务。借助 CloudAudit，您可以记录日志、持续监控并保留与整个腾讯云基础设施中操作相关的账号活动。CloudAudit 提供腾讯云账号活动的事件历史记录，这些活动包括通过腾讯云管理控制台、API 服务、命令行工具和其他腾讯云服务执行的操作。这一事件历史记录可以简化安全性分析、资源更改跟踪和问题排查工作。

### 简化的合规性

借助操作审计 CloudAudit，您可以自动记录和存储腾讯云账号中已执行操作的事件日志，从而简化您日后会进行的合规性审核。

### 用户与资源变更的可见性

操作审计 CloudAudit 可通过记录腾讯云管理控制台操作和 API 调用来提高用户和资源活动的可见性。您可以识别调用腾讯云服务的账号、发起调用的源 IP 地址以及执行调用的时间。

### 安全性分析和问题排查

借助操作审计 CloudAudit，您可以通过捕捉特定时段内您腾讯云账号中所发生更改的全面历史记录，发现并解决安全性和运行问题。

- 查看账号活动
- 持久化存储账号活动记录

### 业务类型

完整保存账号操作记录，进行合规追溯和安全审计。

### 使用建议

借助跟踪集功能，您可以将账号下发生的所有活动信息持久化存储到 COS，从而简化日后您可能进行的合规性审核。
