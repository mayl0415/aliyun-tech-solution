# 地域管理系统

地域管理系统（Region Management System）是管理腾讯云地域资源的统一工具。腾讯云用户可通过统一的地域云 API 获取各个云产品的地域信息，再通过地域信息操作相应资源；对高阶用户还可统一管控账户内的地域和可用区，有效提升用户运维体验。

操作简单

通过云 API 操作，极大简化维护地域信息的操作成本。

自定义配置

支持用户以产品维度、账户维度去自定义地域可用区的启用和停用状态。

标准统一

腾讯云所有产品都按统一的方式对外公开地域。

实时获取

帮助您清晰的了解每个地域可用区的各种属性，轻松获取地域上支持的产品情况。

- 地域可用区
- 自定义地域

用户可自行管理账户下资源所在区域，既可按产品模式管理地域和可用区；也可按账号模式管理。对于无需经常使用的地域，可全局禁用，避免资源管理混乱。[查看已支持该功能的产品](https://cloud.tencent.com/document/product/1596/94989)
