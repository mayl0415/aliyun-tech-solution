# Web 应用托管 Webify

Web 应用托管（CloudBase Webify，webify）为您的 Web 应用提供一站式托管服务，支持包括静态网站、动态 Web 服务等各种类型的 Web 应用，提供默认域名、自定义域名、HTTPS、CDN 加速，提升 Web 应用的性能和安全性。

### 免费默认域名

Web 应用托管为每个应用分配一个专属的默认域名，天然支持 HTTPS，您也可以将自己的域名绑定至 Web 应用托管。

### 前端框架集成

Web 应用托管集成了各类流行的 Web 开发框架，从 CSS 和 HTML 文件组成的简单静态网站，到 React、Vue 等框架构建的 Web 应用，都能完美支持。

### 加速您的站点

Web 应用托管为每个应用默认提供 CDN 接入能力，可根据您的应用资源类型进行缓存，无论您的用户身处何处，都可快速加载您的应用资源。

### 持续集成和部署

Web 应用托管内置 CI/CD 能力，您只需要将变更推送至 Git 仓库，便可触发应用的构建和重新部署。

### Git 平台集成

Web 应用托管集成了多个代码托管平台（Github、Gitlab、Gitee 码云），可直接导入现有仓库，快速创建应用。

### 按量计费，轻松开始

Web 应用托管按照应用的实际资源使用量计费，极大节约时间和资源成本，同时节约额外的服务运维投入成本，对初创期的项目十分友好。

- 静态网站
- 单页面应用（SPA）
- 服务端渲染（SSR）应用

Web 应用托管对诸多开源 SSR 框架进行了适配，如 Next.js、Nuxt.js，配合底层 Serverless 的计算能力，让您摆脱繁杂的配置和运维工作，轻松托管您的 SSR 应用。

### Web 应用托管和云开发 CloudBase 的关系是什么？

Web 应用托管基于云开发 CloudBase 提供的基础资源（静态托管、HTTP 访问服务、云函数等）为开发者提供应用层面的能力，每个 Web 应用在创建时，将会创建一个与之一对一绑定的云开发环境，并使用此云开发环境的资源提供服务。
