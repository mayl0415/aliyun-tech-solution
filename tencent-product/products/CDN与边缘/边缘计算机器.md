# 边缘计算机器 ECM

边缘计算机器（Edge Computing Machine，ECM）通过将计算能力从中心节点下沉到靠近用户的边缘节点，为您提供低时延、高可用、低成本的边缘计算服务。ECM 按实际使用量计费，您可以根据业务需求调整边缘模块服务区域和规模，迅速灵活的应对业务变化，低成本为用户提供更快速的响应。

### 就近服务

边缘计算机器的资源建设在全国各省市靠近用户的边缘数据中心，覆盖移动、联通、电信等主干网络，提供低网络时延、大带宽的计算和网络云服务，为时延敏感业务提供更优质的用户体验。

### 云边协同

边缘计算机器可与腾讯云服务器实现云边协同，使业务充分利用中心-边缘计算的协同优势。业务在向未来演进时，可以更快捷的将云端能力下沉至边缘。

### 安全可靠

边缘计算机器的实例运行在逻辑隔离的私有网络中，切实保证您云上资源的安全性。同时可以提供基础 DDoS 防护和云镜主机防护、木马检测、暴力破解防护、漏洞扫描等基础或专业防护。

### 高效运维

边缘计算机器提供了功能全面的控制台和云 API 接口，您可以结合自身运维需求，使用控制台或 API 来实现对边缘模块和实例的管理、操作等，并可以结合免费提供的云监控能力，获取各项监控指标。

### 弹性灵活

边缘计算机器按实际使用量计费，支持在几分钟内完成模块和实例的调整，如创建、退还等。您可以根据业务需求波动情况，随时灵活调整模块规模。

### 成本优势

边缘计算机器支持按需创建使用资源，根据业务波动灵活调整用量。您可以将每分投入都用在业务需求上，避免初期规模化的投入和业务波动时的空耗，节省大量资本投入和占用成本。
