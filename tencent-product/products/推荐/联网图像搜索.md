# 联网图像搜索

联网图像搜索（Web Image Search，WIMGS）为您提供互联网公开的图像信息的搜索技术服务。您可以通过 API 接口调用本服务，本服务将针对您输入的搜索关键词，检索并返回相关图片的 URL 等多维信息。

### 精准图片匹配

基于文本与图像的相似度智能匹配，返回相关图片 URL 等信息，确保搜索结果高度相关，用户只需提供简洁的关键词即可快速获得精准的搜索结果。

### 多平台丰富内容源

汇集全网可公开索引的网页信息，确保搜索结果覆盖面广、信息来源丰富。

### 实时快速响应

联网图像搜索服务通过可扩展的分布式存储系统，支持毫秒级低延时的实时返回，确保用户体验流畅。

### 多条图片结果召回

每次查询支持最多返回20条相关图片结果，帮助用户获取更多选择，满足不同场景需求。

### 多维度图片信息返回

提供缩略图 URL、原图 URL、尺寸及标题等详细数据，开发者可根据自身业务需求灵活选择使用。

### 便捷集成

采用腾讯云标准云 API 设计及 CAM 鉴权能力，用户开通服务后获取标准化 SDK，即可完成部署开始调用，快速集成至业务应用中。

### 什么是联网图像搜索？

联网图像搜索（Web Image Search，WIMGS）以 API 接口形式为您提供互联网公开的图像信息的搜索技术服务，您可以通过 API 接口调用本服务，本服务将针对您输入的搜索关键词，检索并返回相关图片的 URL 等多维信息。
