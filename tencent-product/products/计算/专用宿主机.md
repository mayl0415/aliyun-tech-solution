# 专用宿主机 CDH

专用宿主机（CVM Dedicated Host，CDH）提供用户独享的物理服务器资源，满足您资源独享、资源物理隔离、安全、合规需求。专用宿主机搭载了腾讯云虚拟化系统，购买之后，您可在其上灵活创建、管理多个自定义规格的云服务器实例，自主规划物理资源的使用。

![](https://qcloudimg.tencent-cloud.cn/raw/b8dea6cb1c715d393033201650d3b0a0.png)

### 资源独享

专用宿主机提供物理机级别的资源独享，您可以自主规划宿主机内资源的使用，避免其他租户的资源竞争。通过腾讯云的控制台或 API 接口，您在几分钟内就能快速购买到专用宿主机。

![](https://qcloudimg.tencent-cloud.cn/raw/6c11a6ceb2ef242361d2ffe737863939.png)

### 安全合规

CPU、内存、磁盘、网络资源均单租户专用。物理机级别资源隔离，提供敏感业务数据保护、磁盘消磁能力，满足金融行业强监管需求。

### 自主灵活

用户可以在指定的专用宿主机上分配云服务器自主规划宿主机资源的使用。实例规格支持自定义，灵活配置，打破子机规格的限制，保障业务性能的同时充分利用物理服务器资源。

![](https://qcloudimg.tencent-cloud.cn/raw/b806aac22fa0589ecad1f30b31af971d.png)

### 具有 CVM 特性

独享子机即专用宿主机上创建的云服务器 CVM，提供镜像、安全组、配置调整、SSH 密钥等支持，各功能特性的使用方式也保持跟普通 CVM 一致。

![](https://qcloudimg.tencent-cloud.cn/raw/a92096318cfc36bc9f95da6757decf86.png)

### 轻资产

按需购买，分钟级交付，腾讯云统一标准的运维管理服务，保障您的资源稳定运行，客户无需关注底层运维，节省人力、节约运维成本，专注于核心业务。

- 金融业务场景
- 高性能业务场景

高性能业务场景、场景描述、游戏直播等对性能稳定性要求较高的业务场景，使用专用宿主机能够保障物理服务器资源的独享，用户自行规划专用宿主机上业务的部署，在充分利用资源的同时避免资源抢占。

### 专用宿主机的计费方式？

可通过包年包月的计费方式以独享宿主机资源的方式购买专用宿主机。购买专用宿主机后可在宿主机上自由创建 CVM，且无需再为 CVM 的 CPU、内存、本地盘进行付费。
