# 批量计算 Batch

批量计算（BatchCompute，Batch）是为有大数据计算业务的企业、科研单位等提供高性价比且易用的计算服务。批量计算 Batch 可以根据用户提供的批处理规模，智能地管理作业和调动其所需的最佳资源。有了 Batch 的帮助，您可以将精力集中在如何分析和处理数据结果上。

  Batch 目前已全面开放，您可以登录控制台提交计算作业和管理计算集群](https://console.cloud.tencent.com/batch/job?rid=1)
- [#### Batch 新上线计算环境

  快速创建、一键扩缩、支持作业提交，搭配高性价比批量计算型资源（0.1元/核/小时）](https://console.cloud.tencent.com/batch/env?rid=1)
- [#### 快速创建任务模板

  将常用的任务制作成任务模板，基于任务模板定制不同的任务，实现作业的快速提交](https://console.cloud.tencent.com/batch/task?rid=1)

### 完全托管

用户可将批处理计算中的资源调度和流程调度完全托管给批量计算。用户定义和提交计算作业后，只需关注批量计算的计算作业状态。用于计算的云服务器创建和销毁、计算环境的部署和执行、原始材料和结果的存储管理，完全由 Batch 来处理。

![](https://qcloudimg.tencent-cloud.cn/raw/247d8efa69f34c6a65b02a031cd732cc.png)

### 成本优化

Batch 可以根据任务的数量和资源的要求，动态配置和弹性扩展用户批处理任务而创建的计算资源，降低资源的使用成本，同时 Batch 零预投成本，用户可以以更低的开销，更充分地利用资源，完成计算作业。

### 强大易用

Batch 提供了大量辅助功能，帮助用户定义和提交计算作业，如多种执行方法定义、计算流程编排、状态监控等。深度结合其他相关云服务，如对象存储 COS。批量处理从计算材料获取、计算作业提交和进行、计算结果上传三个大环节上提供了一站式闭环服务。
