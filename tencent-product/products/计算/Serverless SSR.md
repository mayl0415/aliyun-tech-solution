# Serverless SSR

基于 Web Function 部署 SSR 框架，支持接收与处理原生 HTTP 请求，无需再做事件格式转换，原生 Web 框架仅需几行代码，即可平滑迁移上云。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/-c8WD8svrBTZl8lt2Zq8G.png)

### 低延迟

Serverless SSR 基于 Web Function 部署，直接接收原生 HTTP 请求，无需事件请求格式转换，提升处理效率。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/TEUpipYiKQLmKeAnEEZs6.png)

### 平滑迁移

Web Function 保留了本地监听端口启动方式，原生 Web 框架仅需几行改造，即可平滑迁移上云。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/uQ6FmiQuxERvN5Ryt7r4r.png)

### 简化配置

Serverless SSR 方案无需复杂配置，通过部署 Web Function 云函数，即可高效、快速构建您的 SSR 应用，基于云函数提供的多项高级能力，助力业务上线。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/b05i0xQv2SZbVW03_UZj6.png)

### 降低成本

Serverless SSR 基于网关和 Web Function，支持根据流量的波峰波谷自动扩缩容，无需关心底层资源的管理，减少手动配置，降低运维复杂度，节省使用成本。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/LmKoX6TO2S_tOoW5y7CWi.png)

### 监控排障

云函数提供了丰富的监控排障能力，您可通过查看运行日志与可视化监控图表，实时监控管理您的 SSR 应用。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/NMrlYdxW_yt3Z6U8VDBos.png)

### 安全可靠

通过 SSL 加密，保证 API 的安全通讯；通过可靠的鉴权方式，实现对来源访问权限的 token 认证，保证业务的安全性。

### Serverless SSR 目前支持哪些 SSR 框架?

目前已支持基于 React 的 next.js 框架和基于 Vue 的 nuxt.js 框架。除此之外，您还可以通过 egg.js. express 等组件轻松搭建您自己的 SSR 框架项目。
