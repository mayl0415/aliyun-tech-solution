# 云 HDFS

云 HDFS（Cloud HDFS，CHDFS）为您提供标准 HDFS 访问协议，您无需更改现有代码，即可使用高可用、高可靠、多维度安全、分层命名空间的分布式文件系统。 只需几分钟，您就可以在云端创建和挂载 CHDFS，来实现您大数据存储需求。随着业务需求的变化，您可以实时扩展或缩减存储资源，CHDFS 存储空间无上限，满足您海量大数据存储与分析业务需求。此外，通过 CHDFS，您可以实现计算与存储分离，极大发挥计算资源灵活性，同时实现存储数据永久保存，降低您大数据分析资源成本。

### 易于使用

通过提供标准 HDFS 访问协议，让您使用起来就像在 Hadoop 分布式文件系统（Hadoop Distributed File System）中管理和访问数据；同时 CHDFS 支持开发人员和用户所熟知的文件系统语义，用户在迁移业务到云时无需学习新的存储范例。

### 卓越性能

通过提供分层命名空间，实现原子目录操作，满足大数据分析中频繁的目录创建、目录复制和目录删除操作；在海量大数据处理时提供优异的存储性能，满足您大数据存储与分析业务性能需求。

### 多维度安全

通过支持权限 ACL，实现授权地址和访问类型控制；通过支持 VPC 方式，实现网络访问隔离；同时接入 CAM，实现不同账号授权，满足客户安全且精细化的管理需求，为您访问数据保驾护航。

### 稳定可靠

腾讯云 CHDFS 提供达99.9%的服务可用性和99.999999999%的数据可靠性，其底层采用基于分布式多副本存储策略，保证了数据在任一副本出现故障时快速进行迁移和恢复。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/nb2O25x-CklM3HYJJLqKY.png)

### 成本最优

在腾讯云上您可以在几分钟之内快速扩展或缩减存储资源，以满足变化的大数据业务需求，您可以确保所使用的 CHDFS 在需求高峰期存储空间无限扩展，在需求平淡期进行缩减，以节省成本。

### 服务集成

腾讯云 CHDFS 支持与其他腾讯云产品联动，包括弹性 MapReduce EMR、云数据仓库套件 Sparkling、云服务器 CVM 等，合力在大数据计算、存储方面为用户业务提供完善的解决方案。

在海量大数据分析中，腾讯云 CHDFS 可以结合腾讯云弹性 MapReduce（EMR），同时也可以结合客户在 CVM/CPM2.0 上搭建 Hadoop 大数据分析，实现从 HDFS 上云零成本、计算与存储分离，并快速构建和部署分析应用程序。

### 可以跨地域访问 CHDFS 吗？

不能，目前只支持在 VPC 网络内，通过同地域访问 CHDFS。
