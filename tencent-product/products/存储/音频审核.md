# 音频审核 AAS

音频审核（Audio Auditing Service，AAS）是数据万象产品下的子服务，能够对云存储中的音频进行有效识别，识别出可能令人反感、不安全或不适宜内容的音频并处置，协助用户建设绿色健康的社交网络环境。

### 高效可用

服务毫秒级响应，每秒超过万级并发，支持海量的数据样本，可快速动态扩容，灵活应对高并发需求，无需担心性能损耗。

### 方便易用

支持 API 调用和控制台配置，可视化页面操作，详尽的操作指南，增量内容自动触发，帮助您快速接入使用，免除繁杂的开发成本。

### 功能丰富

支持多种音频格式，提供丰富全面的模型学习样本，覆盖各个行业数百种类型，用户可根据实际业务按需适配。

### 成本最优

使用音频审核，您无需前期投资，无需人力维护，自动化智能识别为您大幅度降低自建人工审核成本。

### 业界领先

依托腾讯多年积累的安全经验，可精准识别并音频中的内容，服务亿万用户产品，在外打破多项世界纪录，技术处于业界领先位置。

### 持久可用

高性能的缓存层，搭配高性能的处理集群，快速返回音频审核结果。针对较大音频处理场景，增加了异步处理队列逻辑，提高审核成功率。

### 音频审核可以处理哪些格式的音频？

音频审核目前支持处理 mp3、wav、aac、flac、amr、3gp、m4a、wma、ogg、ape 格式的音频。
