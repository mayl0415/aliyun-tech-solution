# 文件存储

观看产品视频介绍

文件存储（Cloud File Storage，CFS）为您提供安全可靠、可扩展的共享文件存储服务。文件存储可与腾讯云服务器、容器服务、批量计算等服务搭配使用，为多个计算节点提供容量和性能可弹性扩展的高性能共享存储。腾讯云文件存储的管理界面简单、易使用，可实现对现有应用的无缝集成；按实际用量付费，为您节约成本，简化 IT 运维工作。

### 集成管理

文件存储兼容 POSIX 接口 ，可跨平台访问，并保证文件数据的一致性。用户的 CVM 实例可通过标准 NFS 协议访问 文件存储文件系统。

### 简单易用

文件存储提供简单、易于入门的控制台界面，您可以轻松快捷地创建、配置和管理文件系统，为您节省自建 NAS 存储的部署及维护工作。

### 共享访问

多台 CVM 或容器服务可以共享一个文件存储文件系统，运行在不同可用区下的计算节点也可以通过 VPC 网络使用同一文件存储文件系统，实现多计算节点的协同工作及数据共享。

### 弹性扩展

文件存储文件系统的存储容量可弹性扩展、且不影响应用或服务。同时，文件系统的性能会随存储容量线性增长，提供高可靠、高性能的服务。

### 按需付费

文件存储可以动态分配容量而无需提前预购，您只需按实际使用量付费。文件存储的可多端共享特性也使用户无需重复购买其他的存储。

### 安全可靠

文件存储标准文件存储为 3 份冗余，具有极高的可用性和可靠性。文件存储可以通过用户隔离，网络隔离，以及来访白名单来限制客户端的操作权限。

- 企业文件共享
- 流媒体处理
- Web 服务及内容管理
- 大数据应用

文件存储具备大数据应用程序所需的规模和性能，其计算节点高吞吐量、写后读一致性以及低延迟文件操作的功能及特性，特别适合服务器日志集中处理和分析。
