# 流计算 Oceanus

流计算 Oceanus 是大数据产品生态体系的实时化分析利器，是基于 Apache Flink 构建的企业级实时大数据分析平台，具备一站开发、无缝连接、亚秒延时、低廉成本、安全稳定等特点。流计算 Oceanus 以实现企业数据价值最大化为目标，加速企业实时化数字化的建设进程。

### 一站开发

以 WebIDE 和云 API 两种方式提供集图形化开发、多语言开发、元数据管理、代码调试、依赖管理、部署管理、运行监控、指标告警、智能诊断、黑窗诊断等为一体的一站式开发运维平台。

![](https://qcloudimg.tencent-cloud.cn/raw/e5ae1fbbc7934e19388977131383005c.png)

### 无缝连接

无缝对接消息队列、数据库、数仓等腾讯云主流数据产品和自建开源大数据组件，按需拓展 Connector 对接各类外部数据系统；100%兼容 Apache Flink，支持自建平滑迁移上云。

![](https://qcloudimg.tencent-cloud.cn/raw/ed8f7b15c9c06a3773c41d535ed31055.png)

端到端亚秒级数据处理延迟，单核每秒数十万条记录处理能力，同时支持数万并发超大规模实时任务计算。

### 什么是 CU？

CU（Compute Unit）是流计算 Oceanus 所提供计算资源的最小单位，1CU 的具体含义为： CPU1核; 内存4G。 计费的标准就是用户实际的 CU 使用量。
