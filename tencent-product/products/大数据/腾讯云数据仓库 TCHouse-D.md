# 腾讯云数据仓库 TCHouse-D

腾讯云数据仓库 TCHouse-D（Tencent Cloud TCHouse-D）基于业内领先的 OLAP 数据库 Apache Doris 内核构建，兼容 MySQL 协议，融合云上大数据生态，提供丰富的集群管控能力及完善的巡检告警体系，为客户提供简单易用、轻松运维的云上全托管服务，助力客户快速进行实时 OLAP 数据分析。

[产品文档](https://cloud.tencent.com/document/product/1387/52624)

![](https://qcloudimg.tencent-cloud.cn/raw/762c21ef421ebe4256b2bc2b0f1ed852.png)

### 开箱即用

通过控制台十分钟内可完成分析集群构建，提供完善的集群运维管理、监控告警等功能，使您无需关注底层基础设施，只需点击鼠标即可完成创建集群，修改配置，设置告警等常用功能。

![](https://qcloudimg.tencent-cloud.cn/raw/c0d5f8fbf334a643dc67a14e8bce8cf8.png)

### 简单易用

支持标准 SQL 语言，兼容 MySQL 协议，用户使用标准 SQL 即可开始数据分析，无需理解不同数据设施的数据构造，无需学习新的编程语言，使用简单。

![](https://qcloudimg.tencent-cloud.cn/raw/24d49ce0997ce4d5549b08f40b7b75b9.png)

### 高并发查询

合理的架构设计，灵活的资源分配策略，轻松支持每秒几万甚至到十万级的并发查询，有效支持数千用户同时进行数据分析。

![](https://qcloudimg.tencent-cloud.cn/raw/bec9d8a28e3f550d503890019d86422f.png)

### 性能卓越

全新的向量化执行引擎，强大的 MPP 执行框架，支持星型模型和雪花模型，针对 PB 级数据量的分析仅需亚秒级响应时间即可获得查询结果。

![](https://qcloudimg.tencent-cloud.cn/raw/67ae12f853afd9572663dc60b8a2cb92.png)

### 弹性伸缩

通过控制台简单操作就可以快速实现的集群的扩容，缩容，节点的 CPU 内存配置变更，稳定快捷的支持业务发展，减少成本浪费。

![](https://qcloudimg.tencent-cloud.cn/raw/c06a4cf54ad9fba9c2ff894d4babaa9b.png)

### 安全可靠

双节点同步冗余，实现用户无感的故障转移和容灾备份。分布式部署，计算单元、服务器、机柜三重防护，提高重要数据基础设施保障。用户集群独立部署，支持 VPC 隔离，数据访问安全多重保障。

- 多维商业分析
- 用户行为分析

在网站、App 和游戏中，对用户的点击，时长等用户行为数据进行收集，导入到腾讯云数据仓库 TCHouse-D 中，借助腾讯云数据仓库 TCHouse-D 的优秀查询性能，分析系统进行多维度，多模式分析时可以在亚秒级内响应，快速分析出用户行为特征和规律，为精准营销和会员转化等业务提供强力支持。
