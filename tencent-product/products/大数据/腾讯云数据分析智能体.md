# 腾讯云数据分析智能体

腾讯云数据分析智能体（Tencent Cloud Data Agent）提供面向企业数据自主分析与洞察的智能体服务。通过检索和分析多模态数据，主动进行思考、洞察与行动决策，加速释放企业数据价值

### 数据融合

连接多种数据源，融合多模态数据。TCDataAgent 自主协调结构化与非结构化数据，实现跨系统、跨场景无缝协作，为业务提供全面洞察

### 智能交互

支持用户使用自然语言交互，TCDataAgent 精准拆解业务需求，自动执行检索、分析和报告生成等任务，提升数据价值转化效率

### 业务理解

TCDataAgent 支持用户配置语义知识，自动理解业务语境和业务规则，智能关联用户数据与业务目标，实现数据深度分析与洞察

### 深度洞察

多 Agent 自动实现从任务执行到洞察分析和行动决策的闭环。TCDataAgent 从业务互动中不断学习积累，提升理解能力和准确性

- 智能数据分析
- 企业知识助手
- 智能会话助手

智能会话

腾讯云数据分析智能体自动识别用户意图，检索知识信息，生成沟通话术，提升服务效率和客户体验

- 分析 Agent 自动构建用户画像查询流程，实时展现客户画像
