# 游戏反作弊 ACE

游戏反作弊（ACE），是基于10余年的经验沉淀，倾力打造的一站式游戏安全解决方案。覆盖游戏反外挂、游戏加固、内容安全、经济安全等多种安全服务，专业构筑游戏安全防线。

### 业内核心技术

基于腾讯多年的技术积累，采用机器学习、人工智能等核心技术，时刻保护游戏安全。

### 专业安全团队

行业专业的游戏安全专家团队，基于十余年经验积累，专业护航游戏安全。

### 全场景游戏安全方案

丰富的功能矩阵，覆盖手游、端游、单机游戏等多种游戏类型，在游戏全生命周期内持续提供安全产品支持。

### 全面兼容性

应用于腾讯内外部数百款精品游戏，兼容2000余种设备，覆盖多种架构、引擎及运行模式，保护数亿游戏玩家。

### 海量外挂渠道

专业的外挂监控方案和团队，确保外挂更新渠道的覆盖。

### 能力开放

支持用户进行自主策略管理、运营策略自定义、实时反外挂能力拓展等。

游戏反外挂

提供漏洞挖掘、外挂样本分析和检测以及视觉异常类和行为异常类检测相结合的全链路、多维度、多层级的反外挂能力。

游戏加固

提供代码加密、反调试、反篡改、进程保护等专业防护能力，有效提升客户端被非法第三方软件篡改的难度，为游戏安全保驾护航。

内容安全

基于海量游戏样本数据及文本检测技术和人工智能技术，有效识别游戏中出现的色情、低俗、辱骂、广告、拉人等违规内容，净化游戏环境。

经济安全

通过客户提供的授权接口，基于客户提供的游戏数据，为游戏开发商、游戏运营商等客户提供游戏黑产角色风险识别的技术能力。

### 场景描述

游戏产业中普遍存在的恶意打金、薅羊毛、恶意拉人、恶意广告、色情诈骗等黑产行为，严重破坏游戏经济系统，影响玩家正常体验，占用服务器资源。

### 解决方案

通过客户提供的授权接口，基于客户提供的游戏数据，为游戏开发商、游戏运营商等客户提供游戏打金号、仓库号、点券金币商、资源商、礼包商、号商等游戏黑产角色风险识别的技术能力；基于前置拦截或准入门槛提升，整体减少黑产对游戏运营造成的伤害。

### 手游加固方案会针对 Java 层文件加密吗？

加固方案不会对 Java 层文件进行保护，其原因有：

游戏将自己的逻辑核心代码放在 Java 层的情况几乎不存在，大部分游戏的核心代码和逻辑都在 so 文件。

加固方案是一套游戏专用壳，需要降低对游戏性能的影响。

从游戏稳定性方面考虑，加固专注于 native 部分的核心代码。
