# 运维安全中心（堡垒机）

腾讯云运维安全中心（堡垒机）（Operation and Maintenance Security Center (Bastion Host)）可为您的 IT 资产提供代理访问以及智能操作审计服务，为客户构建一套完善的事前预防、事中监控、事后审计安全管理体系，助力企业顺利通过等保测评。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/e8rFRhw4vYafID3Dl1V0p.png)

### 全面的资产管理

支持多云、线下资产的统一管理，支持主流的操作系统和数据库版本，有效辅助管理员进行安全运维。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/8m3lgZPIY3qyLpowRiGyR.png)

### 细粒度权限控制

支持基于用户、资产、账号、操作权限等维度进行细粒度授权，确保用户所拥有的权限是其访问资产、完成工作任务的最小化权限。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/xmDYD-31-TmA7YB2HJ8gB.png)

### 良好的运维体验

支持多种主流运维操作客户端工具，并且适配 Windows 和 MacOS 终端，运维过程当中，不改变运维人员原有的操作习惯。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/NRir5MTr8__XlK7U_F3uV.png)

### 运维操作审计

对运维人员访问服务器的操作命令、传输文件和运维过程进行审计，确保安全事件有效追溯。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/pZ5NlkxFE2U94BaxpVxId.png)

### 自动化运维

无需在服务器上安装 Agent 即可实现批量、自动化执行任务，提升运维效率。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/dDM09sA3NlogcdsyPON82.png)

### 智能账号管理

支持账号推送和改密计划，协助管理人员对资产账号进行管理

### 通过运维安全中心连接无法连接主机，应该如何排查问题？

先检查录入运维安全中心的账号密码是否正确，然后再检查网络是否连通，如果是设置了安全组，或者是运维安全中心所在 VPC 与 CVM 所在 VPC 不同时，会由于网络不通导致堡垒机无法访问主机。如果是 Windows 主机，可以检查访问用户是否在 Remote Desktop Users；如果是 Linux 主机，可检查访问用户是否允许密码访问。
