# T-Sec 数据安全审计

腾讯云数据安全审计（Data Security Audit）是一款基于人工智能的数据库安全审计系统，可挖掘数据库运行过程中各类潜在风险和隐患，为数据库安全运行保驾护航，是企业的等保合规利器。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/LpmuQLrfL3B3woOOO9y3d.png)

### 全面资产覆盖

基于 SaaS 化架构，可跨地域、跨 VPC 进行统一的资产审计管理。支持公有云、混合云等多种资产场景，支持云数据库、自建数据库、大数据审计，实现全面的资产覆盖。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/SAnLuh9YsHqKuSI0bvoff.png)

### 全量数据审计

基于会话的全量审计，超越传统安全审计概念，能够全量捕捉数据库中的SQL操作。提供了全面的审计类别和可扩展的存储空间，便于操作行为详细溯源，为数据安全事件提供了便捷的追责依据。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/sErANdC1xFwBlo7xmrldd.png)

### 智能威胁识别

依托于腾讯云的专业深度学习算法模型和海量样本训练环境，智能引擎能够应对多变的攻击场景，监控各类变体攻击以及非常见的威胁操作。在数据攻击日益多样化的今天，数据安全审计为数据库安全提供了更精准的威胁分析和告警。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/ilxYHgCyE8owlOmDVE_Qb.png)

### 多种风险告警

提供多种告警方式，包括短信、邮件和企业微信。当检测到威胁时，能够及时通知安全管理人员，以便迅速采取措施，在威胁造成严重影响之前，有效遏制并消除潜在风险。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/23wq16_QfI1My69Peuv7f.png)

### 定期统计报告

数据安全审计支持周期性统计报告，有利于提高管理员对全局安全信息的掌控力度。同时，产品提供专业的等级保护报告，能够更清晰、更有条理地展示符合等级保护相关规定的安全措施，为您通过等级保护测评提供支持。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/MaNIAJKKpg4iX_Ivkm2PJ.png)

### 超高性能吞吐

具有高性能网络引擎和 SQL 语句分析引擎。单节点可支持 30000 条 SQL/秒级吞吐量，1000 万/小时入库速率，基于 TKE 技术可轻松应对更高并发。12 亿条在线 SQL 语句存储能力可轻松实现海量数据检索与分析，还原数据安全事件全貌。

- 危险操作感知
- 安全事件追责
- 性能监控
- 安全信息统计
- 混合云部署

企业的业务往往采用混合云部署方式，即同时部署在腾讯云、其他云及 IDC 的两个或多个位置。腾讯数据安全审计能够帮助企业对位于多个位置的数据资产纳入统一的审计管理，实时感知其安全风险并预警，为企业的业务安全运行保驾护航。
