# 腾讯特效 SDK

腾讯云视立方·腾讯特效 SDK（Tencent Effect SDK）（美颜特效 SDK）是音视频终端 SDK（腾讯云视立方）的重要组成部分，基于优图精准的 AI 能力和天天P图丰富的实时特效处理，为各类视频处理场景提供了丰富、实时、高效的美化能力。腾讯特效 SDK 支持与腾讯云直播 SDK 、短视频 SDK、音视频通话 SDK 等音视频终端产品集成，高效便捷，优势尤为明显。

### 自然的美妆效果

通过图像融合、图像增强、精细形变控制等技术，可以实现实时美颜美型、智能美妆，打造出最丰富自然的美颜效果。

### 稳定的动效贴纸

基于腾讯影像能力中台多年的技术沉淀，动效贴纸稳定跟随面部关键点，3D 能力趋于行业领先，轻松实现沉浸式贴纸互动体验。

### 应用场景广泛

腾讯特效 SDK 支持与直播 SDK、短视频SDK、音视频通话 SDK 联合集成，全面覆盖直播、视频通话、短视频等常见音视频场景。

### 个性化素材创作

提供配套专业特效素材制作工具，海量素材，模板丰富，支持客户自定义2D/3D贴纸，个性化的美妆素材设计，导入 SDK 中即可使用。
