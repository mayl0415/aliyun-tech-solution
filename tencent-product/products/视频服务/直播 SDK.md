# 腾讯云视立方·直播 SDK

腾讯云音视频提供完善直播服务：在提供直播推流和观看的基础上，集成连麦、弹幕、送礼等丰富功能，并配有完善的 UI，帮助实现更快速、便捷的直播接入。

[Demo 体验](https://rtcube.cloud.tencent.com/component/experience-center/index.html#/detail?scene=live)

- [产品公告直播 SDK 产品升级公告](https://cloud.tencent.com/announce/detail/2167)

直播功能展示

![直播开播与观看](https://cloudcache.tencent-cloud.com/qcloud/ui/static/portal_advantage_new/app.card1.151963.png)

### 直播开播与观看

主播可在直播前预览画面，支持开启和关闭直播；支持多人同时观看直播

![单房间连麦](https://cloudcache.tencent-cloud.com/qcloud/ui/static/portal_advantage_new/app.card2.6e37d8.png)

### 单房间连麦

支持单房间内与多位观众进行连麦互动

![跨房连线/PK](https://cloudcache.tencent-cloud.com/qcloud/ui/static/portal_advantage_new/app.card3.7e2599.png)

### 跨房连线/PK

支持双人、多人同时跨房连线/PK

![聊天弹幕](https://cloudcache.tencent-cloud.com/qcloud/ui/static/portal_advantage_new/app.card4.0a6059.png)

### 聊天弹幕

用户可发送表情和文本消息弹幕，实时展示于直播间内

![直播预览](https://cloudcache.tencent-cloud.com/qcloud/ui/static/portal_advantage_new/app.card5.88afde.png)

### 直播预览

观众无需进入直播间，可在直播间外预览直播间内容

![直播瀑布流](https://cloudcache.tencent-cloud.com/qcloud/ui/static/portal_advantage_new/app.card6.0300f0.png)

### 直播瀑布流

观众可上下滑动查看不同的直播间，支持单列或双列瀑布流 UI

![送礼点赞](https://cloudcache.tencent-cloud.com/qcloud/ui/static/portal_advantage_new/app.card7.4c599b.png)

### 送礼点赞

提供弹幕、全屏 (SVGA格式) 及点赞动画，展示礼物面板并支持配置礼物信息

![PC推流助手](https://cloudcache.tencent-cloud.com/qcloud/ui/static/portal_advantage_new/app.card8.37c92b.png)

### PC推流助手

支持多机位画面采集与本地音视频混流，适用于 PC 端专业推流

![](https://cloudcache.tencent-cloud.com/qcloud/ui/static/portal_advantage_new/8182b30f-bb42-4c83-981b-f7837d2c7c6b.png)

### 开发成本低

提供完整的直播解决方案，涵盖主播端与观众端的全套组件和用户界面，有效降低开发投入，显著优化开发成本

### 极速接入

具备完善组件和 UI，无需自主搭建，可实现一天集成、迅速上线，极大提高开发效率

### 功能丰富

支持弹幕、送礼、点赞、观众连麦等互动功能，以及观众管理、观众标记、美颜、背景音乐等配置功能

### 全球实时连接

基于优化的网络架构，确保在全球范围内都能享受低延迟、高质量的互动体验

### 超低延迟

高质量音视频输出，低延迟（低于300ms），确保流畅的直播互动体验，带来沉浸式视听享受

![](https://cloudcache.tencent-cloud.com/qcloud/ui/static/portal_advantage_new/b95746f6-baa8-4fdd-b408-7bea23842a2a.png)

### 至臻画质

采用“至臻画质”技术，在设备性能不足或带宽受限的弱网环境中，能够有效减少视频卡顿并提升画面清晰度，确保流畅且高清的直播体验
