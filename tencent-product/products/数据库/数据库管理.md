# 数据库管理

数据库管理（Database Management Center，DMC）是一个高效，安全，可靠的数据库一站式管理平台。DMC 提供可视化的库管理、实例会话管理、SQL 窗口、SQL 安全审计、SQL 变更审批、实时监控、操作审计等数据库管理能力，集成诊断优化和数据可视化分析能力，从而简化和规范数据库管理操作、降低数据库运维门槛、提升运维效率。DMC 现已推出 2.0 新版，融合了原个人版与企业版，并提供数据库 AI 相关能力。

### 集多功能于一体

集库管理、SQL 窗口、SQL 审批、数据管理、用户授权、实例监控、实例会话、操作审计等功能于一体，提升用户管理数据库的效率。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/jRjcppWVm-EMYlOMdxBj8.png)

### SQL 安全审计

支持对 SQL 语句进行审核，并对不规范的 SQL 语句提供优化建议，降低 SQL 注入风险。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/VDkW15MrhQRLZUZfFb55X.png)

### 数据变更审批

支持通过提交工单对数据库进行 SQL 变更，发起变更后进行 SQL 安全审计和工单审批，保证 SQL 输入的规范性和安全性。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/2WJp1aW4F_0iC2uuETKBX.png)

### 实例会话管理

支持按用户、访问来源、数据库、活跃会话数、运行中会话最长时间等多维度实时统计实例会话，并支持按条件 Kill 会话，方便用户查看和结束目标实例的会话。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/qGgc5mMGm8Eld-9yWJP5Y.png)

### 多维度实时监控

支持实时采集数据库实例的状态数据、基本性能数据和业务性能数据，并通过图表或视图展示，以便用户快速获取监控信息。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/9nhM06KorTeY_oay7ylcd.png)

### 支持多种数据库

支持接入多种关系型数据库、NoSQL 数据库及分布式数据库。

- 统一的数据库管理平台
- SQL 审批

### 场景描述

用户手动输入 SQL 变更数据库数据时，希望对 SQL 进行审批，避免因 SQL 不规范和安全问题对整个数据库系统造成安全隐患。

### 应用建议

数据库管理提供 SQL 安全审计和工单管理能力，用户提交 SQL 数据变更后，对 SQL 语法进行安全审计和 SQL 变更工单审批，保证 SQL 输入的规范性和安全性。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/u4CEt9OuyjV8lQlODi7FJ.svg)
