# 云数据库 TencentDB for PostgreSQL

### 功能强大

基于 SQL 支持 JSON、XML、Key-Value 等非结构化数据类型，丰富的 SQL 语法及函数支持；基于 Synchronous Replication 复制技术，实现商用级的数据强一致，PostgreSQL 已成为商用的首选开源关系型数据库。

![](https://qcloudimg.tencent-cloud.cn/raw/562a9608f2dd93d4e2ba96b50453d992.png)

### 性能卓越

云数据库 PostgreSQL 适用于 OLTP 或 OLAP 场景的高性能数据库，具有与商业数据库媲美的查询优化器，支持所有主流多表连接查询；基于 NVMe SSD 存储，最大 QPS 可达23W，使您以更少的数据库数量支撑更高的业务并发请求量。

![](https://qcloudimg.tencent-cloud.cn/raw/cc6405d09c58b81c552b7e7d08d280a3.png)

### 便捷管理

PostgreSQL 数据库默认支持备份、恢复、监控、迁移等全套解决方案，您只需在腾讯云管理中心点击操作，便能便捷管理。大量性能视图，可查看正在执行 SQL 、当前锁等待、表扫描和索引扫描方面等性能数据，帮助您更快速精准地定位性能问题。
