### 如何使用腾讯混元生图？

腾讯混元生图是一款提供 AI 图像生成与处理能力的 API 技术服务，使用流程包括如下步骤，更多指引详情可参见 [快速入门](https://cloud.tencent.com/document/product/1668/87020)：

（1）注册腾讯云账号，通过实名认证。

（2）登录腾讯混元生图 [控制台](https://console.cloud.tencent.com/aiart)，开通服务。

（3）通过 API 和 SDK 命令行来使用服务，可以通过 [API 3.0 Explorer](https://console.cloud.tencent.com/api/explorer?Product=aiart&Version=2022-12-29&Action=TextToImage) 进行在线测试，或通过 API 3.0 Explorer 生成服务端 SDK 代码，结合腾讯云已编写好的开发工具集（SDK）来调用腾讯混元生图 API。

（4）付费使用 API 或在 [控制台创作页](https://console.cloud.tencent.com/aiart/image-to-image-create) 通过可视化界面使用 API 服务。
