# 移动解析 HTTPDNS

移动解析 HTTPDNS 基于 HTTP 协议向腾讯云的 DNS 服务器发送域名解析请求，是面向多端应用，包括移动端APP/PC客户端应用的域名解析服务，可以避免域名劫持和跨网访问问题，实现精准调度，解决移动互联网服务中域名解析异常带来的困扰。

### 全网域名解析

支持全网域名解析，包括腾讯云/DNSPod，任意第三方注册域名。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/gEIqf3t8if-Y-pTJ-4iyF.png)

### 智能化调度

直接获取到用户 IP，利用自有专利技术生成的 IP 地址库以及测速系统，确保请求访问到最优的节点。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/P-Bb7DuUYvyXt6J5xWo0y.png)

### 安全防劫持

域名解析请求直接发送至 HTTPDNS 服务器，避免域名劫持和跨网访问，实现精准调度，支持 AES/DES/HTTPS 加密方式，查询过程更安全。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/o0eWnpQImwKVim6lztbAW.png)

### 快捷接入

通过 API/SDK 实现快速接入，兼容 IOS/Android 多平台移动应用。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/H0LA7TUkgfgY4eNBFVBxQ.png)

### 解析联动加速

面向权威解析托管在 DNSPod 的域名有加成的缓存加速能力。

![](https://qcloudimg.tencent-cloud.cn/trisys/assets/product/images/QK_5HZbfD8_mwjley1doA.png)

### 稳定可靠

服务可用性不低于99.99%，确保域名解析服务稳定可靠。

移动 App 基于 SDK 快速开发，使用 HTTP/HTTPS 协议将域名解析请求发送到腾讯云的 HTTPDNS 服务器，以获得准确的解析结果。支持多种加密方式，从而保证信息传输安全。在异常情况时，仍旧可以切换为 DNS 协议向运营商 Local DNS 发起解析请求。对于域名在 DNSPod 解析的域名，还可以实现数据联动，做到域名解析调整秒级生效。
