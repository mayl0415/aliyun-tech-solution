### 智能解析

单机解析性能高达 1 亿 QPS，支持 Anycast 全球节点部署，提供更稳定、高可用的解析服务。

### 超强防护

具备 10 TB+ 带宽储备，强大高防能力，轻松抵御大流量攻击。

### 流量调度

支持负载均衡，可任意分配权重。提供各种维度的细分线路，并支持自定义线路及分组。

### 秒级部署

权威解析与递归解析数据同步，流量调度实时生效，实时同步至递归服务器。

### IPv6 支持

全面支持 IPv4 及 IPv6 双协议栈，满足最新行业网站接入规范。

### API 开放

提供丰富的 API 接口服务，自有业务可定制接入、深度融合，让 DNS 管理更加灵活。
