# 日志数据解决方案

腾讯云日志数据解决方案提供采集、存储、检索、统计分析等服务，协助企业方便管理和使用日志数据，满足多种日志场景需求。

[立即申请](https://cloud.tencent.com/apply/p/vi5fj9634if)

### 方案优势

![](https://qcloudimg.tencent-cloud.cn/raw/f988325f9c4dc0519d9d3b8eb5f95f6b.svg)

融合腾讯云优质的产品生态，提供日志采集存储、内容结构化，极速检索、多维分析的一站式解决方案。

![](https://qcloudimg.tencent-cloud.cn/raw/2ab73dbe800e841f1420f3bcc6f1f51f.svg)

采用高可用的分布式架构存储，对数据进行多冗余备份存储，为日志提供稳定可靠的服务保障。

![](https://qcloudimg.tencent-cloud.cn/raw/6bc71a867a44834261ea2819cef1f30b.svg)

支持每日 MB-TB 的日志数据量的采集存储；支持 PB 日志检索秒级返回结果，无需关心扩缩容，轻松应对海量日志。

![](https://qcloudimg.tencent-cloud.cn/raw/c88fe2019151db83cbfbd6856429d693.svg)

无需基于如 ELK 的开源框架从零搭建，无复杂环境依赖，五分钟即可完成配置，接入日志数据。

### 满足不同日志场景

- 后台运维问题定位
- 终端用户行为分析

后台运维问题定位

![](https://imgcache.qq.com/open_proj/proj_qcloud_v2/gateway/solution/log-data/css/img/topu_01.svg)

## 客户面临的挑战

- 如何统一检索庞大集群中上百台后台服务器的日志。
- 如何在日志数据量很大的情况下迅速定位问题。
- 如何对统一存储日志数据并进行生命周期管理。

## 对应的解决方案

- 在后台服务集群上统一安装日志服务（CLS）提供的自研 Agent-LogListener，即可将日志集中采集到用户的日志服务空间。
- 使用日志服务控制台的检索页面，通过异常关键词可快速检索出异常事件的日志，比如通过查询 traceid 将异常事件的调用信息完整筛选出来，同时也支持模糊查询和范围查询，方便业务运维和快速定位问题。
- 通过日志服务提供的投递能力，可以将日志数据投递到对象存储（COS）或其他云上产品，享受集中式的日志数据存储与生命周期管理。

终端用户行为分析

![](https://imgcache.qq.com/open_proj/proj_qcloud_v2/trunk/project/gateway/solution/log-data/css/img/topu_02.svg)

## 客户面临的挑战

- 如何收集安卓、iOS、APP、网页的多种终端的应用日志。
- 如何有效统一存储不在终端本地落地的日志。
- 如何快速实施分析终端用户行为。

## 对应的解决方案

- 基于日志服务平台提供 API 与丰富的终端 SDK，支持用户在终端应用快速植入SDK，锚点采集日志。
- 通过投递对象存储（COS）实现终端日志的集中存储。
- 基于 EMR 的大数据分析能力，对投递至对象存储的日志进行分析，理解终端用户的行为，并结合云图可视化分析结果，助力商业决策。
