# 对象存储内容审核

提供图片、视频、音频、文字、文档等多媒体文件的内容审核能力，支持数据上云自动审核、云上历史数据批量审核。

![](https://cloudcache.tencent-cloud.com/qcloud/tcloud_dtc/static/portal_advantage_new/81f8a6e9-9d03-4db1-b4eb-6b7a27131fa4.png)

### 性价比高

按实际使用量收取费用，无需前期投资，无需人力维护，自动化智能识别为您大幅度降低自建人工审核成本。

### 功能丰富

支持增量数据自动触发，存量数据批量审核，违规数据自动冻结，审核模型覆盖各个行业数百种违规类型。

### 快速接入

支持 API 调用与控制台配置，可视化页面操作，增量内容自动触发，无需开发，帮助您快速接入使用。

### 高效可用

服务毫秒级响应，每秒超过万级并发，支持快速动态扩容，灵活应对高并发需求，无需担心性能损耗。

### 方案概述

客户将业务数据存储在对象存储 COS 上，开启内容审核服务后，可自定义配置检测场景、自定义回调阈值、开启自动冻结等。增量数据自动触发，服务毫秒级响应，对上传文件进行评分，评分越高风险越大；存量数据通过任务形式进行大规模批量审核。审核结果可在控制台可视化展现，客户可根据返回的结果信息进行文件冻结等进一步操作，规避违规风险。
