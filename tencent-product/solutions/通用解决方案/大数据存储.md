# 大数据存储解决方案

通过 COS 和 CHDFS，实现计算与存储分离，发挥计算弹性与存储分层，降低分析成本。

### 易于使用

通过提供标准 HDFS 访问协议，客户使用起来就像在 Hadoop 分布式文件系统（Hadoop Distributed File System）中管理和访问数据。

![](https://cloudcache.tencent-cloud.com/qcloud/tcloud_dtc/static/portal_advantage_new/5d19827a-97c3-4f07-8312-8a6fc63b4ccc.png)

### 卓越性能

通过提供分层命名空间，实现原子目录操作，满足大数据分析中频繁的目录创建、目录复制和目录删除操作，在海量大数据处理时提供优异的存储性能。

![](https://cloudcache.tencent-cloud.com/qcloud/tcloud_dtc/static/portal_advantage_new/bc0d2289-5ecc-479e-9526-a9989531b06d.png)

### 成本最优

在几分钟之内快速扩展或缩减存储资源，以满足变化的大数据业务需求，确保存储资源在需求高峰期无限扩展，在需求平淡期进行缩减，以节省成本。

![](https://cloudcache.tencent-cloud.com/qcloud/tcloud_dtc/static/portal_advantage_new/92baeede-6cda-492b-80f0-732c8787a4b4.png)

### 服务集成

支持与其他腾讯云产品联动，包括弹性 MapReduce EMR、 流计算 Oceanus 、云服务器 CVM 等，合力在大数据计算、存储方面提供完善的解决方案。

### 方案概述

在海量大数据分析中， 对象存储 COS 和云 HDFS 可以结合弹性 MapReduce EMR，同时也可以结合客户在 CVM/CPM2.0 上搭建 Hadoop 大数据分析，实现从 HDFS 上云零成本、计算与存储分离， 极大发挥计算资源弹性伸缩能力，同时对存储数据冷热分层，降低大数据分析资源成本。
