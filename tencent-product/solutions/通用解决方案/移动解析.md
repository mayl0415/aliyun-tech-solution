# 移动解析解决方案

解决移动互联网中的域名解析问题，根治域名劫持（ DNS 劫持），精准识别来源请求，令您的用户访问最准确的服务节点。

解决移动互联网中的域名解析问题，根治域名劫持（ DNS 劫持），精准识别来源请求，令您的用户访问最准确的服务节点。

[查看接入指引](https://cloud.tencent.com/document/product/379)

### 为您解决什么问题？

## 解决域名劫持问题

HTTPDNS 基于 HTTP 协议向腾讯云 DNS 服务器发起域名解析请求，替代基于 DNS 协议向运营商 LocalDNS 发起解析请求的传统方式，可以避免域名劫持、解析不精准和流量跨网的等问题，从而解决用户在使用移动APP时的访问异常。

## 提升服务稳定性

移动解析解决方案接入了 BGP Anycast 网络架构，与全国 Top 17运营商建立了 BGP 互联，确保全国各地的用户请求都能够快速访问到 HTTPDNS 服务器，并在华北、华东、华南等多个数据中心部署多个节点，任意节点发生故障均能无缝切换到备份节点，具有高可用性和快速响应能力。
