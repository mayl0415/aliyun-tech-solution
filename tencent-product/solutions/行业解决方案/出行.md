### 解决方案架构

- 设备连接及管理服务方案
- 出行大数据服务方案
- 全局安全保障方案

设备连接及管理服务方案

![](https://qcloudimg.tencent-cloud.cn/raw/0073abe15b0ee7a84467f80995d79ab0.svg)

- 设备接入，基于 MQTT 协议与 IoT Hub 通信，通过发布/订阅的方式实现设备与 Hub，设备与设备之间的通信。
- 设备管理，提供树形结构的设备管理模型，提供设备生命周期的管理等。
- 设备影子，存储设备或应用当前的状态信息，解决弱网环境设备频繁掉线的问题。
- 设备认证和安全策略，为每个设备颁发密码，对通信进行加密，降低安全风险。
- 引擎规则，提供类 SQL 语言的规则引擎，帮助用户将 Topic 中的数据进行过滤处理，可将数据进行 Topic 间的转发，或转发至其他云产品中。
- 通过腾讯物联云实现设备与业务逻辑打通，方便快捷的实现海量设备数据采集、存储、计算以及智能分析等，对业务及设备进行有效指导。

相关产品

- [物联网通信 IoT Hub](https://cloud.tencent.com/product/iothub)
- [云数据库 TencentDB for Memcached](https://cloud.tencent.com/product/cmem)

出行大数据服务方案

![](https://qcloudimg.tencent-cloud.cn/raw/bbcc3c183e5504ee744e17b0a78352e3.svg)

- 用户实时位置等信息，通过 Ckafka 流式处理进行实时数据采集\存储。
- 利用流式大数据工具, 实时处理分析用户行为, 并快速做出路线推荐。
- 实时数据流分析通过 RayData 可视化展现, 可直观的观察实时车辆运行状况与真实用户体验。
- 客户端应用日志以及服务器端运行日志全量存入由对象存储 COS 构成的数据湖中,用于历史数据的批量处理与分析。
- 云中 EMR 集群提供方便的大数据平台搭建模式,快速构建大批量数据分析 Hadoop 集群, 并提供多种大数据生态中的分析工具, 以实现低成本高效率的大规模数据定期批量处理。
- 大数据平台处理后的数据再回存到 COS 作为中间结果, 另一部分清洗结构化之后的数据通过 BI 工具展现。
- 基于数据实时渲染技术，利用各种技术从大规模数据, 实现云数据实时图形可视化、场景化以及实时交互，形成区域热图,智能的车辆/设备投放调度。

- [对象存储 COS](https://cloud.tencent.com/product/cos)
- [大数据可视交互系统 RayData](https://cloud.tencent.com/product/raydata)

全局安全保障方案

![](https://qcloudimg.tencent-cloud.cn/raw/0200fb2b46e48184a62877d95d0fa264.svg)

- 腾讯云 DDoS 防护（大禹）高达300G的防护服务和多达35线的 BGP 线路，让您的业务不再畏惧 DDoS 攻击的挑战，同时拥有极速的访问体验。
- 天御是针对业务安全和内容安全的多重保障，包括活动防刷、验证码反欺诈、登陆保护，让你无后不再担心业务风险。
- Web 应用防火墙通过 Web 入侵防护、0Day 漏洞补丁修复、恶意访问惩罚、云备份防篡改等多维度防御策略全面防护网站的系统及业务安全。
- 云镜基于腾讯安全积累的海量威胁数据，利用机器学习为用户提供黑客入侵检测和漏洞风险预警等安全防护服务，解决当前服务器面临的主要网络安全风险。
- 私有网络 VPC 提供专业的租户隔离，以及包括 ACL 和安全组在内的安全策略，保证云内的网络安全可靠。
- 乐固涵盖应用加固、漏洞扫描、盗版监控、真机测试、质量跟踪、安全支付等服务，配合代码签名证书提供的代码软件数字签名的认证，免除 APP 收到第三方破解和串改的侵害。
- 同时腾讯云还提供专业的安全咨询服务、渗透测试服务等安全解决方案，全面保障您的业务安全合规。
