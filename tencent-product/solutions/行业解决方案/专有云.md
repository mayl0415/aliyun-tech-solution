# 专有云解决方案

基于腾讯强大技术能力和成熟产品体系推出的专有云解决方案，多版本满足不同的客户需求，专业安全稳定的云平台。

[腾讯专有云企业版

腾讯专有云企业版（Tencent Cloud Enterprise，简称 Tencent TCE）是基于腾讯云成熟产品体系自主研发的云平台，提供自主可控、弹性伸缩的全栈服务能力。

- 系统架构：基于腾讯公有云自研架构
- 应用版本：企业版 、大数据版 、AI 版 、敏捷版
- 适用场景：私有云、托管云、行业云、集团云
- 服务客户：大中型国企、央企、集团公司、金融行业

[腾讯专有云 PaaS 平台

腾讯专有云 PaaS 平台（Tencent Cloud-native Suite，简称 Tencent TCS）是基于 Kubernetes 及成熟中间件构建的敏捷 PaaS 平台，帮助企业实现数字化转型。

- 系统架构：基于云原生技术的自研架构
- 交付方式：兼容主流 IaaS、虚拟化或裸金属环境
- 适用场景：SaaS 底座、PaaS 底座、PaaS 平台
- 服务客户：社会企业、政务金融、公共事业

[腾讯云 TStack

TStack 是腾讯推出的企业级私有云解决方案，能够为用户提供私有化 IaaS 服务，对接和支撑 PaaS 及 SaaS 应用，具备安全、高效、稳定、强兼容特性。

- 系统架构：基于 OpenStack 和 k8s 等开源架构
- 交付方式：本地化、黑石、腾讯云 VPC、一体机柜
- 适用场景：政务云、企业云、传统 IT 架构改造
- 服务客户：教育、医疗、企业、金融、政务等
