# 微信云支付解决方案

联合微信支付和服务商协同解决商户端第三方支付的安全性和可靠性问题，助力服务商聚焦业务系统的建设和新业务的探索。

联合微信支付和服务商协同解决商户端第三方支付的安全性和可靠性问题，助力服务商聚焦业务系统的建设和新业务的探索。

### 什么是云支付？

## 云支付是腾讯云联合微信支付推出的移动收单 SaaS 服务，为服务商开展业务提供了易用、低成本、开放和可靠的移动支付收单通道，具有以下优点：

### 低门槛，免开发

- 无需任何软件开发即可帮助商户接入微信支付；支持移动端商户进件；商户资料审核速度更快。

### 费用更低，成本节省80%

- 无需租用服务器及投入人力成本，系统建设及维护费用比自建系统低80%。

### 多种支付方式，开放共赢

- 支持多种第三方支付、自定义广告以及支付后服务商自定义操作，提供会员卡券等等，方便接入微信支付官方活动。

### 安全可靠，可用性99.99%

- 交易和进件信息加密和签名认证，避免商户资金损失；多点多运营商接入，专用支付链路，多机房容灾。

### 丰富服务场景介绍

- 刷卡支付
- 固定二维码支付
- 自建系统对接
- 服务商管理
- 特色增值服务

刷卡支付

![](https://qcloudimg.tencent-cloud.cn/raw/60dc0173d6a62003716328439e3a0d33.png)

云支付提供 Windows 下的收银插件，无需改动自有收银系统，通过先进 OCR 技术采集自有收银软件收款金额后，使用扫码枪扫描顾客二维码即可完成支付。另有云支付简易收银软件，支持基础收银功能，可供商户直接使用。

固定二维码支付

![](https://qcloudimg.tencent-cloud.cn/raw/f8fa5f03cc530cbd9b7fcc379723ffa4.png)

服务商自动开通，无需开发即可为商户的每个门店生成一个收款码，顾客使用微信/支付宝扫描二维码，输入金额即可进行付款。支付相关页面可展示服务商自定义广告、logo 等进行营销推广，收款码支持动态绑定。

自建系统对接

![](https://qcloudimg.tencent-cloud.cn/raw/479e913652ac20dcf4f274ebdbe39091.png)

针对服务商已有自建收银终端或收银后台的场景，云支付提供 Windows 下的 SDK 及 HttpsAPI 接口，方便有技术研发能力的服务商（如专业的收银软件厂商）接入云支付，帮助服务商解决底层的安全问题，聚焦于上层业务逻辑。

服务商管理

![](https://qcloudimg.tencent-cloud.cn/raw/a518f00c0a1f4b0b4509857db9992920.png)

服务商可通过云支付查询子服务商/子商户交易情况、下载账单、分级管理子服务商。可在云支付上设置代理商，通过代理商进行商户进件等操作。提供微信支付活动报名入口，方便服务商帮助商户快速参加微信支付各种营销活动。

![](https://qcloudimg.tencent-cloud.cn/raw/6f1e1602651d1210dd7f1e394e524431.png)

### 多种接入方式介绍

- 直连模式
- 相关系统接入

直连模式

![](https://qcloudimg.tencent-cloud.cn/raw/e570516544144b96e2999738954ea807.svg)

普通服务商可通过云支付直连微信支付等第三方支付，服务商需在微信支付等第三方支付平台申请商户号，再将商户信息录入到云支付，即可接入。

相关系统接入

![](https://qcloudimg.tencent-cloud.cn/raw/cbd30c886efd158bef06074b7e1c1aa9.svg)

云支付提供了开放的接口，方便整个支付环节中的各个子系统提供商接入，接入后，云支付会向服务商推荐解决方案。
