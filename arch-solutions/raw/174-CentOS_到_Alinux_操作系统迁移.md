## 通过操作系统迁移解决企业问题

**简单易用**

白屏化操作，控制台可查看迁移进度和迁移日志。

**数据安全保障**

无需手动创建快照，系统自动创建快照保障。原系统盘会被释放，新系统盘保留原有数据，无需担心数据丢失。

**低成本**

原地升级不需要额外的资源轮转和人力成本，分钟级迁移。

## CentOS 到 Alinux 操作系统迁移

![](https://help-static-aliyun-doc.aliyuncs.com/assets/img/zh-CN/6745113271/p832194.png)

通过在专有网络 VPC 创建云服务器 ECS，并将云服务器 ECS 导入服务器迁移中心 SMC 或 ECS 控制台中的实例详情，可以将云服务器 ECS 的操作系统从源操作系统（CentOS）一键迁移为目的操作系统（Alibaba Cloud Linux/Anolis）。

**部署时长：**40 分钟

**预估费用：**1 元（假设您选择 1 核 2G 规格的 ECS，资源运行不超过 1 小时，且体验完后删除 ECS 快照。实际可能会因您操作过程中使用的资源规格和流量差异，导致费用有所变化，请以控制台显示的实际报价和最终账单为准）

## 技术方案的广泛应用场景

  ### CentOS 7/CentOS 8 停止维护

  CentOS 官方宣布了停止维护 CentOS Linux 的计划，操作系统迁移可将阿里云 ECS 云服务器操作系统从 CentOS 7/8 平滑迁移到 Alibaba Cloud Linux 和 Anolis 操作系统，提高您更换 CentOS 操作系统的效率。

  ### Alibaba Cloud Linux 2 EOL 升级

  操作系统迁移可将阿里云 ECS 云服务器操作系统从 Alibaba Cloud Linux 2 平滑升级到 Alibaba Cloud Linux 3，享受更新版本的体验。
