## 通过弹性公网 IP 解决企业问题

弹性公网 IP（EIP）的优势在于提供稳定、独立的公网 IP 地址，支持快速 IP 切换和灵活的带宽调整，确保业务连续性和 IP 稳定性。

**灵活管理 IP 资源**

EIP 具有更高的灵活性和可调整性。与固定公网 IP 不同，EIP 可以轻松地与不同的资源（如 ECS 实例）进行关联和解绑，便于快速调整和管理 IP 资源。

**快速服务切换**

当需要迁移业务或当一个可用区发生故障时，可以将 EIP 更绑到另一个资源上快速实现业务切换或故障屏蔽，不需要重新进行域名解析，提高了服务的稳定性，减少业务中断时间。

**降低带宽成本**

EIP 支持包年包月、按固定带宽和按使用流量计费等多种计费策略，且支持通过共享流量包和共享带宽有效降低带宽成本。

## 技术方案的广泛应用场景

  ### 业务迁移

  当用户的业务需要从一个 ECS 迁移到另一个 ECS 时，可将原先绑定在旧实例上的固定公网 IP 转化为 EIP，并将其绑定到新的 ECS 上。

  ### 水平扩容

  当需要升级到更高配置的 ECS 或增加多个 ECS 实例时，用户可将固定公网 IP 转化为 EIP，并将其绑定到负载均衡 SLB 或 NAT 网关上。

  ### 故障逃逸

  当 ECS 实例发生故障，用户可创建一个新的 ECS，并将固定公网 IP 转化为 EIP，并绑定到新的 ECS 上，以实现快速故障逃逸。
