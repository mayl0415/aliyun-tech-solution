## 利用 Cypher 打造图数据分析解决方案

**传统关系型数据库的图计算困境**

在处理高度互联数据时，传统关系型数据库需要多次 JOIN 操作，导致查询效率低下，难以满足实时分析需求，尤其在社交网络、推荐系统等场景中。

**图数据库的建模与扩展挑战**

专用图数据库虽然适合处理关联数据，但往往面临数据规模扩展性差、运维复杂、与现有系统集成困难等问题。

**PolarDB 图引擎实现高效分析**

PolarDB PostgreSQL 版集成图计算引擎，支持 OpenCypher 查询语言，提供高性能图遍历能力，同时结合云原生数据库的弹性伸缩与高可用特性，轻松实现图分析。

## 高效挖掘关系网络，显著提升洞察与决策效率

### 兼容 Apache AGE

PolarDB PostgreSQL 版高度兼容 Apache AGE 图引擎，全面支持知识图谱的存储与高效查询。标准 ANSI SQL 与图查询语言 openCypher 可在同一数据库集群中无缝协同，实现关系型数据与图数据的混合分析，兼顾灵活性与执行效率。

### 统一的图形和关系型查询

允许同时处理关系型数据和图形数据，支持在同一个查询中混合使用 SQL 和图查询语言，使得处理复杂的数据模型更加容易和高效。

### 弹性扩展与高可用

采用云原生架构，支持存储与计算分离，可秒级扩展计算节点，自动处理故障转移，保障业务连续性。

### **按需付费与免运维**

采用 Serverless 模式，用户只需为实际使用的计算资源付费，显著降低运维成本。开发者无需关心底层计算资源的运维和管理，让企业更加专注于核心业务系统。
