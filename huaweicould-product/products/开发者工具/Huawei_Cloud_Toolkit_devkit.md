# 什么是Huawei Cloud Toolkit

华为云开发者插件（Huawei Cloud Toolkit），作为华为云围绕其产品能力向开发者桌面上的延伸，帮助开发者快速在本地连接华为云，打通华为云到开发者的最后一公里。Huawei Cloud ToolKit是围绕华为云开发者生态工具体系打造的一系列IDE插件，支持[VS Code](https://code.visualstudio.com/download)、[IntelliJ IDEA](https://www.jetbrains.com/idea/download/?spm=a2c4g.98762.0.0.49521a88JfVjg6#section=windows)等系列平台、以及[华为云自研 CodeArts IDE](https://devcloud.cn-north-4.huaweicloud.com/codearts/download) ，帮助开发者更高效、便捷的搭建应用。致力于为开发者提供更稳定、快速、安全的编程体验。
