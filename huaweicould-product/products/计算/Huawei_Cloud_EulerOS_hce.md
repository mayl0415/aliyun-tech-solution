# 什么是Huawei Cloud EulerOS

#### **定义**

Huawei Cloud EulerOS(简称HCE)是基于openEuler构建的云上操作系统，中文名为华为云欧拉操作系统。

HCE打造云原生、高性能、高安全、易维护等能力，加速用户业务上云，提升用户的应用创新空间，可替代CentOS、EulerOS等公共镜像。
