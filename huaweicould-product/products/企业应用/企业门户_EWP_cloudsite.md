# 什么是企业门户？

![](https://support.huaweicloud.com/productdesc-cloudsite/public_sys-resources/caution_3.0-zh-cn.png) 

因业务策略调整，华为云已于2025年7月26日17:00（北京时间）停售企业门户服务，[查看公告](https://www.huaweicloud.com/notice/2025/20250726165313161.html)，已于2025年12月30日00:00（北京时间）停止企业门户续费和升级，[查看公告](https://www.huaweicloud.com/notice/2025/20250926100531377.html)。

目前企业门户服务不支持新购、续费、升级、复制站点，不支持购买付费模板，不支持购买客户服务包。您已购买的服务在有效期内使用不受影响。如有需求，推荐您根据自己的业务情况使用华为云商店同类产品，详细可查看[云商店官网](https://marketplace.huaweicloud.com/)。

#### 概述

企业门户服务（Enterprise Web Portal Service）是一款致力于网站建设的华为云SAAS服务，无需代码开发，自由拖拽组件，即可快速搭建多终端的网站、网店等。您可使用由企业门户提供的模板自行搭建网站，也可以购买客户服务包后由专业的设计师团队为您量身定制网站。

#### 建站规格

企业门户为您提供了以下几种产品版本，如[表](#cloudsite_01_4081__table899523412254)所示：

**表1** 建站版本

| 产品 | 模板建站 | 定制建站 |
| --- | --- | --- |
| 版本 | 多终端自适应版 | 客户服务包 |
| 规格 | 入门版、标准版、推广版、企业版。 | 基础版、专业版、企业版。 |
| 支持终端类型 | PC、手机、小程序（网页版）。 | PC、手机、小程序（网页版）。 |
| 是否支持终端独立编辑 | 是 | 是 |
| 网页编辑是否支持自适应（一端设计，多端适用） | 是 | 是 |
| 是否包含交易功能 | 仅企业版支持交易功能，其他版本暂不支持交易功能。 | 基于多终端自适应SAAS版本提供对应人工服务 |
| 是否包含知识付费 | 仅企业版支持知识付费功能，其他版本暂不支持知识付费功能。 | 基于多终端自适应SAAS版本提供对应人工服务 |
| 模板是否免费 | 提供3300+免费模板。 | 人工设计网站一次。 |
| 云服务器是否包含弹性公网IP | 不包含云服务器[弹性公网IP](https://support.huaweicloud.com/productdesc-eip/overview_0001.html)。 | 不涉及 |
| 适用场景 | 展示类网站（企业官网类网站）、服务类网站、知识付费类网站等。 | 展示类网站（企业官网类网站）、服务类网站、知识付费类网站等。 |
