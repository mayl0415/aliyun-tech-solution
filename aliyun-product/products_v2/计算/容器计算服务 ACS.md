## 产品简介

阿里云容器计算服务 ACS （Alibaba Cloud Container Compute Service，ACS）是以 K8s 为使用界面供给容器算力资源的云计算服务，提供符合容器规范的算力资源。算力交付模式为Serverless形态，您无需关注底层节点及集群的运维管理。ACS 支持按需弹性和容量预留的资源使用方式，以及秒级的按量付费模式。
### ACS 产品架构
依托于神龙架构，实现容器和资源的一体化。

面向在线业务场景，可使用通用型或性能型容器算力。面向离线业务场景，可使用更经济实惠的 BestEffort 型实例。YAML 文件或控制台简单配置，即可快速创建部署业务。

业内首创算力调配资源复用，用户负载和云产品共享计算资源，提供通用型、任务型和独享型三类 Pod 实例，充分适配多场景、多负载。

## 产品功能

支持通用型、任务型和独享型3种不同Pod实例类型，提供按需弹性、资源预留和节省计划3种付费模式，提供独创的资源配额能力。
客户端、控制台镜像和模板进行应用创建，支持无状态、有状态、任务等多类负载，支持手动伸缩和 HPA 自动伸缩策略，集成阿里云存储和网络产品。
集成 Prometheus 服务、日志服务，支持报警、集群巡检和故障诊断，提供 API Server 的审计日志功能。

## 产品定价

ACS 在业内首创超灵活 Pod 配置模式，支持1:1～1:8的资源配比，屏蔽代际和机型差异，支持秒级弹性、秒级计费，提供按需弹性、资源预留和节省计划三种付费模式，更经济、更从容的应对业务流量波动。
### 计费方式
按量付费是一种先使用后付费的计费方式，阿里云自动按小时从账号中扣除使用资源的费用。 ACS 默认使用按需弹性资源，基于不同实例类型的 Pod 申请资源，秒级按量付费，可以通过账单详情查看对应的计量数据。

提供镜像构建、安全扫描、全球同步和部署的自动化集成应用交付链能力，帮助企业客户构建高效安全交付和自动化安全运维能力，践行安全左移原则。
