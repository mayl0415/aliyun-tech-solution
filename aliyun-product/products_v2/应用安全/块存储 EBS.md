# 块存储 EBS

## 产品简介

 播放视频 块存储 EBS ( Elastic Block Storage ) 是为云服务器 ECS 、容器计算服务 ACS 等提供的低时延、持久性、高可靠的块级随机存储。块存储支持在可用区内自动复制您的数据，防止意外硬件故障导致的数据不可用，保护您的业务免于硬件故障的威胁。

## 应用场景

自建关系型/ NOSQL 数据库 企业核心业务系统 基于快照的数据保护 企业应用系统
场景描述及概要 企业目前通过自建数据库记录和查询用户信息、消息流和状态信息等数据，要求数据库性能必须达到千万 DAU 规模的要求。 场景需求与痛点 DAU 达到千万级别时，IM 即时通信场景的聊天信息和状态信息记录、查询的次数会达到几十万~百万的规模，时延要求必须控制在 ms 级以内。传统的高效云盘和 SSD 盘均无法同时满足需求，需要性能更强的块存储。 解决方案及优势 ESSD 云盘作为自建数据库的块存储，可灵活根据业务要求，选择对应的档位，最高支持 100 万 IOPS 和微秒级时延。 支持即改即生效和弹性伸缩，高性价比满足数据库业务的读写和查询要求。 方案优势与价值 ESSD 云盘以高 IO 和低时延的特性，满足了 IM 场景的数据库性能要求，同时提供多个不同档位选择，让该初创公司研发运维游刃有余。 推荐搭配使用
