# 性能测试 PTS

## 产品简介

作为性能测试工具，性能测试 PTS 支持按需发起压测任务，可提供百万并发、千万 TPS 流量发起能力，100% 兼容 JMeter。提供的场景编排、API 调试、流量定制、流量录制等功能，可快速创建业务压测脚本，精准模拟不同量级用户访问业务系统，帮助业务快速提升系统性能和稳定性。

## 产品优势

- 快速创建场景 支持浏览器、移动端、网关层录场景；支持 har、postman 文件导入
- 灵活编排业务逻辑 零编码编排复杂场景，支持登录态共享、参数传递等指令
- 精准、海量流量发起 动态秒级调整流量，最大支持百万并发/千万 TPS 流量发起
- 支持多协议 支持 HTTP、Websocket、MQTT、RTMP/HLS、微服务、JDBC 等协议
- 流量地域定制 流量发起支持指定阿里云全球地域、运营商，真实反应用户体验
- 全面兼容 JMeter 开源标准 免去 JMeter 运维烦恼

## 产品功能

- 压测场景构建 可对 API 进行串行或并行编排，并定义多种参数。支持 Cookie 传递，提供集合点、思考时间等指令，便于模拟复杂业务场景。
- 流量控制 提供虚拟用户并发和 RPS 吞吐量两种模式，可自动或纯手动秒级调整流量，支持最高千万级的流量瞬时脉冲。SLA 结合定时压测功能可实现无人值守的压测。
- 监控与防护 支持 API 维度的吞吐量、并发能力指标，精度到达秒级别。和云监控、ARMS 监控无缝集成，便于问题定位和流量防护。
- 流量录制 可通过浏览器、移动端对线上环境进行流量录制，抓取操作链路和业务接口请求，进行流量分析并梳理压测模型后得到压测场景。
