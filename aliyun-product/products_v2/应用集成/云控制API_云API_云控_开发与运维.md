# 云控制API_云API_云控_开发与运维

## 产品简介

公测中 云控制 API 云控制 API 是一款免费的OpenAPI开发者工具套件。让开发者能够使用一致的OpenAPI和一个开发者工具包来管理和集成阿里云上百款产品。

## 产品优势

使用一致的API集成阿里云
- 统一的接口 使用统一的 API 动词、入参和一个SDK工具包来进行开发。
- 更快的访问 少量代码更改即可持续集成阿里云新的服务和功能。
- 一致的API规范 使开发人员能够轻松地、一致地管理云基础架构。
- 更低的学习成本 只需学习五个OpenAPI和资源框架，即可持续集成阿里云。

## 产品功能

基础能力支持 可以通过五个基础OpenAPI完成对已支持的阿里云产品进行管理。 即：ListProducts、CreateResource、UpdateResource、GetResource、DeleteResource。
- 基础API能力 ListProducts（查看产品列表）、ListResourceTypes（查看资源类型）、 CreateResource（创建资源）、UpdateResource（更新资源）、 GetResource（查询指定资源）、ListResources（列举资源）、DeleteResource（删除资源）
- 进阶API能力 GetTask（查询异步任务）、ListDataSources（查询资源可选值）
开发者工具包支持 丰富的开发者工具包，适应多语言开发场景
Java 通过Java语言集成与开发
- Python 通过Python语言集成与开发
Go 通过Go语言集成与开发
IDE支持 深度集成多种IDE开发工具，给开发者一站式体验
- VSCode 一站式的VSCode开发工具，可以直接通过快捷命令自动加载资源模型

## 应用场景

面向对象开发 持续集成
使用5个标准化的API，管理阿里云资源 提供稳定、高效、一致的OpenAPI开发者集成体验。针对云服务OpenAPI数量多、业务关系复杂、上下游集成学习成本高进行专项优化，解决开发者集成阿里云过程中维护多个版本SDK，每个云产品一套代码，多个OpenAPI学习成本的现状。通过资源抽象将开发从面向过程升级至面向对象，给开发者提供简洁的开发体验。
能够解决 多OpenAPI维护成本高 通过抽象阿里云OpenAPI资源化，将以前开发者需要集成至少5个OpenAPI才能完成的业务缩减为一个资源。并且可以做到不同产品的代码结构复用。 追求更高开发效率 提供无页面化集成开发体验，创建资源入参均可通过API动态获取。一个SDK集成所有阿里云资源。并在未来支持更加场景化的IDE深度集成。 推荐搭配使用
