## 云数据库专属集群原理

MyBase开放版提供给客户以资源 + 数据库管控 + 数据库引擎 自主搭配的形式购买云数据库服务，目前包括 MySQL、PolarDB X开源数据库，提供丰富的白屏化管理功能，比纯开源自建更灵活、更易用、功能更强大，并赋予主机、数据库所有管理权限，以满足企业客户对自主管理权限的核心需求。

## 产品功能

- 集群创建 通过控制台页面或OpenAPI即可快速创建集群，创建集群时可以自由选择所需的管控组件，如Operator、Orchestrator、Prometheus、ELK、MHA等，集群中部署了管控组件，这些组件便可以应用于创建的数据库实例
- 实例创建 可以快速创建数据库实例，一个集群下可以创建多个数据库实例，包括MySQL、Redis等
- 服务配置中心 服务配置中心提供了一个类似货架的能力，可以在服务配置中心方便的选择安装所需要的管控组件
- MySQL 一个高效、免费的关系型数据库管理系统，适用于各种规模的应用，以其卓越的性能、灵活性和高可靠性支撑着互联网上众多应用程序的数据存储与管理需求
- Redis 基于内存的高性能键值存储系统，支持多种数据结构，常用于缓存、消息队列、数据过期等场景，极大提升了数据访问速度和应用性能
- PolarDB-X PolarDB-X是阿里自研的分布式数据库产品，通过存储与计算分离的架构设计，提供水平扩展能力，旨在应对大规模数据存储与高并发访问场景，保证高性能与高可用性
- 更多组件与引擎 Orchestrator、Prometheus、Grafana、ELK
- 支持DBS 低成本、企业级的云原生备份平台，MyBase使用使用DBS对数据库实例提供企业级备份
- 支持DMS 支撑数据全生命周期的一站式数据管理平台，提供全域数据资产管理、数据治理、数据库设计开发、数据集成（上云迁移、容灾、多活、ETL）、数据开发及数据消费服务等能力
- 支持DTS 数据传输服务（Data Transmission Service，简称DTS），支持关系型数据库、NoSQL、大数据（OLAP）等数据源，提供数据迁移、订阅及实时同步功能
- 支持OSS MyBase的日志数据可以采用OSS作为存储，海量、低成本

## 客户案例

- 小盒科技 在线教育 小盒科技采用云数据库专属集群 MyBase 的方案承载了核心小学业务和网校业务，在专属集群主机之上按需创建出不同的 RDS 数据库实例规格，通过专属集群的超配能力合理配置资源，按业务场景需求快速构建出读写分离、代理短连接优化等不同数据库架构；同时运维上开放数据库和 OS 权限，客户可自主可控运维数据库。
- CC视频 场景化视频 CC视频在阿里云上采用16台专属集群主机承载直播业务场景下消息分发业务。业务高峰来临之前，将核心 Redis 实例使用独占主机部署，独享整机的 CPU 和带宽资源；业务低峰时，通过智能调度整合，通过合理资源超分配，降低整体成本。
- 天冕 金融科技服务 天冕利用专属集群 MyBase 实现主机和数据库的高可用。 社区 PG 9.4.15升级至 AliPG 11.1。 调整 DTS 并发，在不影响业务的同时进行平滑的数据迁移。 利用专属集群的特性，在确保主库资源不被争用条件下，实现资源高效利用。
