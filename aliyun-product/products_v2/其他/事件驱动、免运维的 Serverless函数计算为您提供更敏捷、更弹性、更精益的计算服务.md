## 产品简介

函数计算（Function Compute）是一个事件驱动的全托管 Serverless 计算服务，您无需管理服务器等基础设施，只需编写代码并上传，函数计算会为您准备好计算资源，并以弹性、可靠的方式运行您的代码。 

## 安全合规

函数计算提供内置安全机制帮助用户减轻安全保障的负担，用户需要根据自身实际情况，运用好函数计算提供的安全机制来实现业务安全性。
函数计算通过安全容器实现虚拟机机级别的计算资源安全隔离，且通过 VPC 实现不同租户间的网络隔离和传输保障。
函数计算通过内置的运行时安全、隔离地执行用户代码，由平台负责运行时环境的租户隔离、漏洞修复和安全升级，且升级过程对用户透明。
函数计算通过内置的 3AZ 高可用实现自动容灾，即使某个可用区整体宕机也不影响函数服务，用户无需为冗余资源额外付费。
函数计算的元数据通过 AES256 密文存储，使用时解密后的元数据缓存时间不超过600s，运行时环境提供的缓存在实例释放时随之释放。
函数计算 API 及内部通信均使用 TLS 1.2 及以上协议加密传输内容，函数实例网络访问受限，由用户决定网络外访权限。
函数计算使用隔离的账号缓存代码和层至对象存储，或缓存容器镜像至 ACR 中，除用户自身和服务账号外，任何组织和个人不可访问用户代码。
函数计算通过 RAM 角色和策略精确控制函数访问权限，允许用户以最小授权原则访问其他云服务，且可利用 RAM 策略灵活管理。
